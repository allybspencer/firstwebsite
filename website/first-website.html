Unix Shell Command Cheat Sheet

ls mkdir cd pwd rm echo less tree man
Description Lists files in current directory Makes a new directory Change directory Prints the working directory to tell you where you are Removes a file Prints something to the screen Lets you control how much gets on the screen Shows the beginning of the directory and prints out files underneath Tells you what the command does
Example Command ls mkdir new-directory cd new-directory pwd rm new-directory echo less tree man more
